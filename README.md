## Empezando

Primero, instala las dependencias usando [yarn](https://yarnpkg.com/)

```bash
yarn
```

Segundo, corre el development server:

```bash
yarn dev
```

Abre [http://localhost:3000](http://localhost:3000) con tu navegador.
